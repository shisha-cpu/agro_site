📝 Как использовать

Шаг 1: Убедитесь, что сервер запущен (npm start)

Шаг 2: Загрузите XLSX файл с прайсом или JSON файл (data.json)

Шаг 3: Данные сохранятся в MongoDB и появятся на сайте

💡 Быстрый импорт: Используйте кнопку "Импорт из data.json" для загрузки данных из локального файла
